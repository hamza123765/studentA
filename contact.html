Accessibility Features Applied

  • ✅ Every <input> and <textarea> has a matching <label> with for attribute
  • ✅ <fieldset> and <legend> group related form controls
  • ✅ aria-current="page" on current navigation link
  • ✅ Semantic heading hierarchy: h1 → h2 (no skipped levels)
  • ✅ No presentational <br> tags for layout – structure uses semantic <div> containers
  • ✅ Strict minimal CSS – only color, background-color, font-family, font-size
